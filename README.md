# 🏦 Dashboard BI Bancário Estratégico

Análise de rentabilidade e risco de 5.000 transações bancárias fictícias,
desenvolvido em Power BI como projeto de portfólio.

## 🛠 Ferramentas
- Power BI Desktop
- Microsoft Excel
- DAX (medidas calculadas)

## 📊 Páginas do Dashboard
- **Visão Executiva** — KPIs financeiros e tendência mensal
- **Produtos & Clientes** — rentabilidade por produto e segmento
- **Risco & Agências** — mapa de alertas e ranking de inadimplência

## ❓ 10 Perguntas Respondidas
1. Qual o volume total transacionado?
2. Qual o lucro líquido gerado?
3. Qual o percentual de inadimplência?
4. Qual produto gera mais lucro?
5. Qual produto tem maior margem?
6. Qual segmento de cliente é mais rentável?
7. Existe relação entre score de crédito e inadimplência?
8. Quais agências têm maior inadimplência?
9. Agências físicas ou digitais performam melhor?
10. Qual cidade concentra mais alertas de baixa rentabilidade?
